• Topic

      The topic of this website will be about how to raise a wolf dog in a domestic home.

      This topic will specifically go into the the habits of a wolfdog,

      The website is going to specifically talk about what defines a wolfdog, instructions on raising a wolfdog, and the consequences of having a wolf dog as a companion. The website is also going to contain several real life accounts of people who have experience on raising a wolfdog.

  • Scope/Depth
      Most of the website is going to elaborate on the definition of a wolfdog, and how to raise a wolfdog upon different stages of its life.
  • Target Audience
      This is aimed at an audience who has interest in raising wolfdogs and/or who is already raising a wolf dog.
  • Design, UI/UX
      The website is going to have two informational columns and main content area in the center that is going to host the user interactive panel.
  • How will it be interactive
      A search button is going to be available for the user to search through the wolf database for techniques on raising a wolf dog and pictures of wolfdogs.
  • How will you use programming concepts learned in class:
      The program is going to feature a wolf picture array.

      When the user searches for a wolf, the website is going to loop through the array to look for matching pictures. If the picture is found, the innerhtml of the webpage is then going to be updated to add that picture onto the interactive webpage.
